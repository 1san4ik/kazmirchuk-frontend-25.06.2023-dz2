ДЗ 2. New branch/Pull request/Commit/Merge
  1. створити репозиторій github
  2. клонувати локально
  3. локально створити нову бранчу і переключитись на неї
  4. додати будь-який файл (наприклад index.html)
  5. зробити комміт і потім запушити зміни
  6. перейти в репозиторій github і створити пулл реквест
  7. перейти локально, внести зміни в створений файл, знову комміт, пуш
  8. перейти в репозиторій github і змерджити пулл реквест
  9. перейти локально, переключитись на основну бранчу (main або master) і зробити пулл нових змін
